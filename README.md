# WhTabLayout
仿TabLayout

## 特点
### 1.支持动态添加
### 2.支持水平和竖直方向的展示
### 3.支持自定义tab选中和未选中状态
### 4.支持自定义tabIndicator
### 5.基于RecyclerView，tab重用，占内存更小


## gradle

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.HuiTL2022:WhTabLayout:1.0.0'
	}
